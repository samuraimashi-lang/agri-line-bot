# 圃場のGPS座標の調べ方

fields.json に入力する緯度・経度の調べ方です。

---

## Googleマップで調べる方法（一番簡単）

1. スマートフォンまたはパソコンで **Googleマップ** を開く

2. 圃場の場所をマップ上で長押し（スマホ）
   またはクリック（パソコン）する

3. 画面下部に数字が表示される
   ```
   例：38.3500, 140.3800
   　　↑ 緯度(lat)  ↑ 経度(lon)
   ```

4. その数字を fields.json に入力する

---

## fields.json の記入例

```json
{
  "name":   "荒谷①",      ← 圃場名（アグリノートと合わせる）
  "group":  "天童",        ← 圃場グループ名
  "lat":    38.3500,       ← 緯度（Googleマップで調べた数字）
  "lon":    140.3800,      ← 経度（Googleマップで調べた数字）
  "area_a": 17,            ← 面積（アール）
  "memo":   "メモ（自由）"
}
```

---

## 検知範囲について

現在の設定では圃場の中心から **300m以内** に入ると
圃場として自動判定されます。

広い圃場や複数の圃場が隣接している場合は
担当者に相談して調整してください。
